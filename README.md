## 二、模型选择

目前主流的公式识别模型架构为encoder-decoder，从2021到2024发展的模型如下：

| 模型  | HME100K | 时间 |
| ----- | ------- | ---- |
| BTTR  | 64.1    | 2021 |
| CAN   | 67.31   | 2022 |
| CoMER | 68.12   | 2022 |
| LAST  | ——      | 2023 |
| ICAL  | 69.25   | 2024 |

当然也有Nougat(2023)做过相关工作。

目前主要测试了**CAN**、**CoMER**、**ICAL**、**Nougat**，等模型，测试baseline结果如下：

|        | ExpRate | TokenAcc |
| ------ | ------- | -------- |
| CAN    | 0.34    | 0.79     |
| CoMER  | 0.62    | 0.95     |
| ICAL   |         |          |
| Nougat | 0.66    | 0.98     |
